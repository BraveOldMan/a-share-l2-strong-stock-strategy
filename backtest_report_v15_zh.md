# A股 L2 策略 V15 最终回测报告 (Walk-Forward)

该文档保存了本策略在完成了**夏普平滑收益标签 (Risk-Adjusted LTR)** 与 **等权宽基防火墙 (Market Breadth Filter)** 重装升级后的全周期最终滚动表现评估。我们将此版本定义为 **V15 宽基防火墙大底版**。

## 1. 测试配置与环境
- **模型版本**: V15
- **数据窗口**: 2024-06-05 ~ 2026-03-06 (约 422 个可交易日)
- **滚动验证机制**: Walk-Forward (无未来函数)
- **单轮训练样本 / 步退**: Train = 100 days; Retrain_Every = 20 days; Purge-Embargo = -2 days
- **核心基建参数**:
  - `initial_capital` = 1,000,000 ￥ (100万)
  - `StatefulPortfolioManager` (状态机解耦持仓，不限持有期)
  - 动态跟踪止盈锁 `Trailing Stop` = 8%
  - 绝对止损底线 `Hard Stop` = -5%
- **V15 特性开关**:
  - `Label Target` = `future_close_return + future_max_drawdown * 1.5`
  - `Breadth_MA10_TH` = `-0.003` 强制熔断

## 2. 核心性能统计
以下为完全扣除了单边万五双边万二点五标准滑点与印花税后的净值表现。

- **最终累计净值**: `1,383,532.95 ￥`
- **总回报率 (Total Return)**: **`+38.35%`**
- **年化收益率 (Annualized Return)**: **`+21.39%`**
- **最大回撤 (Max Drawdown)**: **`-8.50%`**
- **夏普比率 (Sharpe Ratio)**: **`0.862`**
- **卡玛比率 (Calmar Ratio)**: **`2.518`**
- **系统胜率 (Win Rate)**: `24.2%` (基于单日结算或单笔盈亏，胜率偏低但受止盈止损截断保护)

## 3. 对照与演进分析 (对比 V12 版本)

| 指标 | V12 (原高抛妖股版) | V15 (夏普平滑稳健版) | 改善幅度 |
|------|-------------------|----------------------|---------|
| 总净值 | 710,967 (-28.9%)  | **1,383,532 (+38.3%)** | 净增逾 67 万 |
| 年化率 | -19.21%           | **+21.39%**           | 逻辑完全反转 |
| 最大回撤| -32.09%           | **-8.50%**            | 降低近4倍量级 |
| 夏普比率| -0.938            | **+0.862**            | 从极差至优秀 |

### 3.1 核心成功要素：
1. **宽基动能拦截生效**: 当大盘等权中位跌落触发阴跌死叉时系统空仓，完美避开了 2026 年初的单边暴跌段，使得长期多头持有策略的全局回撤被牢牢锁死在不足 10% 的极低水位。
2. **标签修正防骗线**: 因 LTR 算法在学习目标中遭受了最大回撤 1.5 倍的严厉抵扣，模型从原先频繁买入**开盘秒冲极值随后诱多跌停**的股票陷阱中挣脱，转而去捕获具有实质护盘并大概率在次日**真正收红**的安全底部形态股。

## 4. 下一步调优展望
针对单发胜率（24.2%）仍然偏低、大量换手均在 -2% ~ -5% 止损滑点的问题，下个版本 (V16) 可以考虑引入更高级别的**大额主动净增买单穿透率**或者**日内 VPIN 绝对值过滤**作为白名单强关联，以此剔除更多“假装护盘”的骗炮股，目标将胜率从 24% 抬升至 40% 即可产生极其惊恐的单边暴利效应。
